# TemplateMCP

独自にMCPを作る時のテンプレートを作るプロジェクト

## 技術スタック

- MCP
    - stdio
        - gRPC
    - 生成AIが操作する
- Next.JS
    - ユーザー(人間)が操作する管理画面
- DB
    - SQLite

## 題材

- Hello world MCP
    - ユーザー(人間)は Web 画面を通して人物と好物を登録できる
    - 生成AIは指示を受けると、全員に挨拶して好物に対するメニューを提案する

## ゴール

- テンプレートに使えるプロジェクトとする
- AI と連携できる
- ユーザーは Web 画面で管理ができる
- 仕様などのドキュメントはdocsに格納する
